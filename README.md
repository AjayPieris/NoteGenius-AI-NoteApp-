NoteGenius is a modern, full-stack journaling application that uses **Generative AI (Google Gemini)** to organize your messy thoughts. 

When you write a raw note, the system automatically:
1.  **Analyzes** the content for sentiment and meaning.
2.  **Generates** a clean, creative Title.
3.  **Summarizes** the note into one sentence.
4.  **Tags** the note automatically for easy categorization.

Built with **Django REST Framework**, **React (Vite)**, **MySQL**, and **Tailwind CSS**.
